# mall 架构设计分析

## 整体架构

`
┌─────────────────────────────────────────────────┐
│                  客户端                          │
│        PC端 / H5 / 小程序 / App                  │
└────────────────────┬────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────┐
│              Nginx 反向代理 + 负载均衡              │
└────────────────────┬────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────┐
│              Spring Boot 应用集群                  │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐         │
│  │ mall-admin│ │mall-portal│ │mall-search│        │
│  └──────────┘ └──────────┘ └──────────┘         │
└────────────────────┬────────────────────────────┘
                     │
┌────────────────────▼────────────────────────────┐
│                   数据层                          │
│  MySQL    Redis    ES    MongoDB    RabbitMQ     │
└─────────────────────────────────────────────────┘
`

## 核心模块

### mall-admin - 后台管理
- 商品管理：SPU/SKU 规格体系、分类、品牌、属性管理
- 订单管理：订单流程、售后、物流跟踪
- 营销管理：秒杀、优惠券、品牌推荐
- 权限管理：RBAC 模型，菜单/角色/用户三级

### mall-portal - 前台商城
- 首页推荐：品牌 + 新品 + 人气
- 商品搜索：Elasticsearch 全文检索
- 购物车：Redis 缓存 + 数据库持久化
- 下单支付：分布式事务保证一致性

### mall-search - 搜索系统
- Elasticsearch 商品索引
- 中文分词 IK 集成
- 搜索建议与高亮

## 关键技术点

1. **JWT 无状态认证** - 前后端分离标准方案
2. **Redis 分布式缓存** - 热点商品、购物车
3. **Canal 数据同步** - MySQL binlog → Redis/ES
4. **RabbitMQ 消息队列** - 订单超时取消、库存异步扣减
5. **Docker 容器化** - 一键部署
